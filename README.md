
# Dodge The Creeps Game

## Trilha de Rust & Godot

Este jogo foi desenvolvido como um projeto final do curso de Rust & Godot oferecido pela **Conway - EACH USP**.

Além da implementação do clássico **Dodge The Creeps**, amplamente utilizado para introduzir novos desenvolvedores à integração entre Rust e Godot, foram adicionadas duas funcionalidades originais, idealizadas e desenvolvidas por mim:
- Mudança da textura de fundo conforme o jogador adquire mais pontos (a primeira mudança aos 10 pontos e a segunda mudança aos 20 pontos);
- Um novo powerup para o jogador, capaz de eliminar todos os inimigos presentes na tela (basta apertar a tecla 'E').

O resultado é uma versão um pouquinho mais expandida do jogo original.
